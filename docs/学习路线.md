# TodoList 扩展版学习路线

此文档为通过 TodoList 项目逐步掌握 Vue 核心与进阶知识点的路线。

---

## 🏁 阶段 1：基础强化

**目标**：巩固 Vue 3 核心概念

**功能扩展**：

- 编辑任务
- 任务计数、过滤器（全部/未完成/已完成）
- 清空已完成任务

**知识点**：

- `<script setup>`
- `ref`, `reactive`
- `computed`, `watch`
- 组件间通信（props + emits）
- 条件渲染、列表渲染
- 双向绑定 (`v-model`)
- 简单过渡动画 (`<transition-group>`)

---

## 🏁 阶段 2：状态管理与全局响应式

**目标**：掌握全局状态管理

**功能扩展**：

- 任务数据保存在全局状态
- 多组件访问和修改任务（例如顶部统计栏单独组件显示任务数量）

**知识点**：

- Pinia（推荐 Vue 3 官方状态管理库）
- 模块化 store（actions, getters）
- 响应式共享状态

**练习**：

- 把 todos、filter 等状态移到 Pinia
- 顶部统计栏和列表组件从 store 获取数据

---

## 🏁 阶段 3：路由与多页面

**目标**：掌握 Vue Router

**功能扩展**：

- 多页面支持：
  - `/`：全部任务
  - `/active`：未完成任务
  - `/completed`：已完成任务

**知识点**：

- Vue Router 基本使用
- 路由参数与导航
- 动态路由与导航守卫

**练习**：

- 将过滤功能由按钮切换改为路由切换
- 用 `<router-link>` 实现导航

---

## 🏁 阶段 4：异步与后端交互

**目标**：掌握 API 调用与异步数据

**功能扩展**：

- 任务存储在后端（模拟 API 或使用 JSON Server）
- 添加、删除、更新任务通过 API

**知识点**：

- `fetch` / `axios`
- 异步方法 (`async/await`)
- 生命周期钩子 `onMounted` 获取数据

**练习**：

- 将本地存储替换为 API
- 添加加载状态与错误处理

---

## 🏁 阶段 5：高级动画与 UX

**目标**：掌握 Vue 的过渡动画与用户体验优化

**功能扩展**：

- 添加删除/新增任务动画
- 编辑模式平滑过渡
- 切换主题动画（淡入淡出）

**知识点**：

- `<transition>` 钩子函数
- JavaScript 动画
- 动态类和样式绑定

**练习**：

- 实现任务列表淡入淡出动画
- 编辑模式过渡动画
- 切换主题时背景渐变

---

## 🏁 阶段 6：插件、组件库和高级技巧

**目标**：熟悉 Vue 生态和可复用组件

**功能扩展**：

- 使用 Element Plus / Ant Design Vue 美化 UI
- 可复用 TodoItem 组件（支持拖拽排序）

**知识点**：

- 插件安装与使用
- 高阶组件、插槽 (slots)
- 提取可复用 composables

**练习**：

- 使用 UI 组件库改造 TodoList
- 封装通用状态管理函数
- 尝试拖拽排序和动态更新

---

## 🏁 阶段 7（可选）：SSR 或部署

**目标**：了解 SSR 与部署

**内容**：

- 使用 Nuxt.js 将 TodoList 改造成 SSR 项目
- 部署到 Vercel / Netlify
- 学习环境变量和构建优化

---

## 💡 建议顺序

1 → 2 → 3 → 4 → 5 → 6 → 7（可选）

每个阶段完成后，都建议写注释、优化性能、调整样式。
通过这个扩展版 TodoList 项目，你几乎可以覆盖 Vue 的所有核心知识点。
